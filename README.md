# 🥗 Health Nutrition App

A comprehensive AI-powered Health & Nutrition mobile application developed using **Flutter**, **Node.js**, **Express.js**, and **MongoDB**. The app helps users maintain a healthy lifestyle by providing personalized meal plans, health tracking, AI assistance, and consultation with certified dietitians.

# 🌐 Environment Variables

Create a `.env` file in the backend folder.

```env
PORT=5000

MONGO_URI=YOUR_MONGODB_CONNECTION

JWT_SECRET=YOUR_SECRET_KEY

FIREBASE_API_KEY=YOUR_FIREBASE_KEY
```
